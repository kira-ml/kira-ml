Im an undergraduate in **Computer Science** with a growing specialization in **Machine Learning**, particularly **Natural Language Processing (NLP)** and **Deep Learning**.  

My path into AI was inspired by the rise of **large language models**, sparking my fascination with how neural architectures capture meaning and structure in human communication.
